# Neon Dash Prototype 🎮

A fast-paced, neon-drenched 2D platformer built in Python using the Pygame-CE engine. Features custom physics, interactive level progression, and a built-in character customizer.

## Features ✨
- **3 Hand-Crafted Levels:** Easy, Medium, and Hard challenges.
- **Character Studio:** Swap neon cube frames dynamically before starting.
- **Visuals:** Ambient background grid arrays and fluid trailing particle vectors.

## How to Install and Play 🚀

1. Make sure you have **Python** installed.
2. Open your terminal and install the game engine dependency:
   ```bash
   pip install pygame-ce
